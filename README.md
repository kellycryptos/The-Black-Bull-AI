# The Black Bull AI 🐂

**Onboard new people into crypto using $ANSEM + Bullpen**

A simple, fun tool that helps people check their $ANSEM holdings, get a shareable allocation card, and discover the Bullpen platform — all in one place.

### Mission
Help onboard 10,000+ new people into crypto through the $ANSEM community and Bullpen platform (which combines everything needed: claims, bullposting, community, and more).

### Features
- Paste Solana wallet + X username → See your simulated airdrop allocation
- Beautiful shareable card with your X profile picture (auto-fetched)
- One-click Download or Copy Image for X
- Live $ANSEM price
- Direct link to claim on Bullpen using my referral
- Beginner-friendly FAQ

### How to Use
1. Go to [the-black-bull-ai.vercel.app](https://the-black-bull-ai.vercel.app)
2. Paste your X username (optional) and Solana wallet
3. Click "Check Allocation"
4. Download/Copy your card and share it
5. Claim your rewards on Bullpen using the link
